# cards-jekyll-theme
## Responsive Column Cards


This site exist from Cards Jekyll Theme - Responsive Column Cards with some features:

   - Clean homepage three cards
   - Related cards post from sub-categories
   - Navigation post
   - Pagination post
   - 404 not found
   - SEO tag
   - Analytics
   - Google adsense
   - Cookie notification
